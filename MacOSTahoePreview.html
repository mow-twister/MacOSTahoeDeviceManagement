macOS Tahoe Upgrade

Available in Self Service for 30 days. After that, the upgrade will be automatically enforced.

This upgrade introduces improvements to security, performance, and usability.

What’s Changing

  • Updated desktop and window design
  • Redesigned System Settings
  • Improved performance and stability
  • Enhanced security and compliance controls

Application Changes

Restricted applications:
  • Messages
  • FaceTime
  • Mail

Use instead:

  • Microsoft Outlook
  • Microsoft Teams

Upgrade Process

  • 30–60 minutes
  • Multiple restarts
  • Device unavailable during upgrade

Before You Upgrade

  • Save your work
  • Connect to power
  • Ensure internet connection
  • Close all applications
